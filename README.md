# score_logistic
构建基于逻辑回归的评分卡模型


主要包括：
  （1）特征工程
  （2）WOE编码
  （3）IV值计算及特征筛选
  （4）特征WOE化
  （5）评分卡建模，输入筛选后的特征的属性值，自动得出评分结果
